- 배열 정렬 함수  **C++의 sort(&arr[0], &arr[N-1], less<T>)와 유사

`Arrays.sort(arr);`

- 기본은 오름차순

```java
import java.util.*;

int [] arr = new int[] {2,3,5,4,1};
Arrays.sort(arr);

//arr : 1,2,3,4,5
```

- 내림차순

```java
import java.util.*;

int [] arr = new int[] {2,3,5,4,1};
Arrays.sort(arr, Collections.reverseOrder());

//arr : 5,4,3,2,1
```

- 일부만 정렬

```java
import java.util.*;

int [] arr = new int[] {2,3,5,4,1};
Arrays.sort(arr, 0, 3);

//arr : 2,3,4,5,1 - 0~4의 index만 정렬
```

- 배열 출력하는 함수

`Arrays.toString(arr)`

- Console에 출력
  
  ```java
  int [] arr = new int[] {2,3,5,4,1};
  
  System.out.println("arr[] : " + Arrays.toString(arr));
  //arr[] : 2,3,5,4,1
  ```
